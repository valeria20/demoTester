# Setting up

1. Install protractor 
   ```npm install -g protractor ```
  
2. Install mocha and chai
   ```npm install -g mocha```
   ```npm install -g chai```
3. Download the necessary binaries
   ```webdriver-manager update```

4. Start up a selenium server 
   ```webdriver-manager start```
    
5. Switch to cloned repo
   ```cd [path]/demoTester```
  
6. Run tests
   ```protractor conf.js```
