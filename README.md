# Setting up

1. Install protractor 
   ```npm install -g protractor ```
   
2.  Switch to cloned repo
   ```cd [path]/demoTester```
  
3. Install mocha 
   ```npm install mocha```

4. install chai
   ```npm install chai```
   
5. Download the necessary binaries
   ```webdriver-manager update```

6. Start up a selenium server 
   ```webdriver-manager start```
    
7. Run tests
   ```protractor conf.js```
